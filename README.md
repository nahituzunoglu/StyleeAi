#  StyleAI

**StyleAI** is an AI-powered outfit recommendation system built using Python and object-oriented programming (OOP) principles. The project simulates a smart styling assistant that generates outfits based on user preferences, events, and seasonal conditions.


##  Features

* Object-Oriented Programming principles:

  * Encapsulation
  * Inheritance
  * Polymorphism
* Clothing item modeling:

  * Shirt
  * Pants
  * Shoes
  * Accessory
* Outfit generation system
* Event-based styling:

  * Work
  * Casual
  * Sport
  * Formal
* Season-based outfit recommendations
* User wardrobe management



##  Tech Stack

* Python 3
* Dataclasses
* Enum
* Object-Oriented Design



##  Project Goal

The goal of this project is to simulate a smart outfit recommendation system by applying object-oriented programming concepts and UML-based design principles. It focuses on clean code structure, modularity, and real-world modeling.
